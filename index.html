Web programozás 1 - Projektmunka

Full-stack webes alkalmazás fejlesztése relációs adatbázis-kezeléssel és modern frontend architektúrákkal. A projekt demonstrálja a kliensoldali állapotkezelés, az aszinkron hálózati kommunikáció és a szerveroldali adatszolgáltatás implementációját.

HTML5 / CSS3 JavaScript (ES6+) React.js PHP 8.x MySQL Git / GitHub

JavaScript CRUD

Natív JavaScript alkalmazás, amely a filmek adatait egy lokális tömbben tárolja. A felületen megvalósításra került az adatok listázása, új filmek felvétele, meglévők módosítása és törlése.

React CRUD

A választott adatbázis fájl adatainak kezelése React környezetben. A megjelenítéshez saját MovieCard komponens készült, az adatok szűrése pedig műfaj alapján történik.

SPA (Single Page Application)

Egyoldalas alkalmazás két különálló React alkalmazással. Egy 4x4-es rácsú Memória játék (useState alapú állapotkezeléssel) és egy Időjárás előrejelző (külső API hívással).

Fetch API (MySQL)

A filmes adatbázis MySQL alapú tárolása. A szerveroldali PHP script (api.php) biztosítja a CRUD műveleteket és egy keresőt, ami kilistázza, hogy az adott filmet mely mozik vetítik.

Axios & React

A React frontend és a MySQL adatbázis összekapcsolása Axios könyvtárral. Az alkalmazás tartalmaz egy mozi-választó szűrőt is a helyszín alapú listázáshoz.

OOJS Snake Játék

Objektumorientált JavaScript alkalmazás. A játék logikája osztályokra (class), öröklődésre (extends) és a super kulcsszó használatára épül egy 20x20-as grid rendszerben.

GitHub Projekt

A forráskód verziókövetése a GitHub rendszerében történt, az elvégzett feladatok a commit message-ekben követhetőek, valamint egy kanban boardot is használtunk a projekt és feladatok nyomonkövetésére.
